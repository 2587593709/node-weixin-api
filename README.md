Weixin API for node

===========

微信公共平台API。


## Install

```sh
$ npm install --save node-weixin-api
```


## Usage

```js
var nodeWeixinApi = require('node-weixin-api');

nodeWeixinApi('Rainbow');
```

```sh
$ npm install --global node-weixin-api
$ node-weixin-api --help
```

```sh
# creates a browser.js
$ npm run browser
```
